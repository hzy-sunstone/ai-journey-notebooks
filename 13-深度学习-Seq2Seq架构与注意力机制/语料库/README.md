# WMT中英机器翻译训练集

## 数据集描述
中英机器翻译训练集，用于训练模型。

### 数据集简介
该数据集来源于世界翻译大会[WMT 2021](https://www.statmt.org/wmt21/translation-task.html) News Translation Task，由ParaCrawl、News-Commentary、Wiki-Titles、UN Parallel Corpus、WikiMatrix、CCMT等数据集组成，共计2500万双语句对。

### 数据集支持的任务
中英双向机器翻译模型训练

## 数据集的格式和结构

### 数据格式
文本格式，source和reference逐行对齐
